weather_script module¶

weather_script.weather_script(init_data_path: str, output_path: str, workers: int, weatherAPI_rpm: int, geoAPI_rpm: int) → None¶
Main function that reads data from csv, processes it,

defines addresses, defines weather in specific area, create plots, save info about weather in cities with the most number of hotels in country

Parameters

  • init_data_path (str) – Path to folder with hotels.zip

  • output_path (str) – Path to folder where you should save results

  • workers (int) – number of threads for parallel processing

  • weatherAPI_rpm (int) – number of requests per minute for weather API

  • geoAPI_rpm (int) – number of requests per minute for geolocation API

Returns

None
